Agent Skills 资料熔炼

  • Claude Blog: Equipping agents for the real world with Agent Skills
  • Claude Platform Docs: Agent Skills overview
  • Agent Skills Open Standard: Agent Skills
  • 参考补充：Specification、Best practices、Using scripts、Optimizing descriptions、Evaluating skills
  1. Skill 是可加载的专业流程包，不是普通提示词、工具或微调。
  2. 渐进披露让智能体先看 name/description，相关时再读 SKILL.md，需要时再读资源文件。
  3. SKILL.md 的 frontmatter 决定可发现性和触发准确性，正文决定执行质量。
  4. scripts、references、assets 分别承担执行逻辑、按需知识和静态模板。
  5. 好 Skill 来自真实任务、真实纠正和真实失败，而不是泛泛最佳实践。
  6. 可靠 Skill 需要触发 eval、输出断言、安全边界和跨平台兼容检查。
