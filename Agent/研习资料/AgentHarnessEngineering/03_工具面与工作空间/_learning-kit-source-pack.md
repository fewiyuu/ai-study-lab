## Working Claims

- 本单元讲的是智能体行动的边界：工具面定义能做什么，工作空间定义在哪里做，状态注入定义谁提供内部字段，输出物边界定义什么算完成。
- 学完本单元，读者应能为一个文件处理、研究摘要或工单处理 Agent 画出工具清单、工作目录、状态字段、错误语义和验收方式。
- 本单元不展开完整协议连接、持久执行、追踪系统或版本迁移策略；这些内容留给后续单元。
