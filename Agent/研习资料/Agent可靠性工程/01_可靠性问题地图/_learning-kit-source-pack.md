## Lesson Boundary

- 本单元只做故障分流和证据定位，不展开完整的恢复实现、幂等实现、重试实现或追踪埋点实现。
- 主锚点建议固定为“客服退款 Agent”。必要时再借“研究 Agent 卡住”做对照。
- 后续单元各自负责的内容：
  - 02 持久执行与检查点
  - 03 确定性、幂等与副作用隔离
  - 04 工具失败、超时与重试策略
  - 05 结构化输出与防漂移边界
  - 06 追踪指标与事件语义
  - 07 评测回放与回归测试
  - 08 上线前验收与运维边界
