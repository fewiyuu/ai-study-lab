# Agent 学习资料索引

这组资料按课程工坊流程制作：当前阶段只生成 `学习页.html`。每一块学完后，在页面底部生成练习记录，把答卷发回来阅卷，再生成对应的归档笔记。

## 学习顺序

1. [01 ReAct 与 Agent 循环](01_ReAct与Agent循环/学习页.html)
   - 目标：理解 Agent Loop 为什么是“思考 → 行动 → 观察”，以及 ReAct 如何把推理和工具调用交错起来。
2. [02 Agent 工程架构](02_Agent工程架构/学习页.html)
   - 目标：把 Agent 从一次聊天理解成可运行系统，能描述规划、工具、记忆、权限、日志和失败恢复。
3. [03 Agent 设计模式](03_Agent设计模式/学习页.html)
   - 目标：识别常见工作流模式，能判断什么时候用 routing、planning、reflection、tool use、human-in-the-loop 等设计。
4. [04 评测追踪与安全](04_评测追踪与安全/学习页.html)
   - 目标：知道 Agent 不能只看最终回答，要能用 task、trial、grader、trace 和安全边界做验收。
5. [05 Multi-Agent 系统设计](05_MultiAgent系统设计/学习页.html)
   - 目标：判断什么时候真的需要 Multi-Agent，并能设计角色、上下文边界、协作协议和验证方式。

## 使用方式

- 按顺序逐个学习，不要一次性跳到后面。
- 每页学完后完成 20 道练习，并点击“生成练习记录”。
- 把生成的练习记录发回来，我会按题目阅卷、指出盲区，再生成最终归档笔记。
- 如果某页读完仍然说不清“我现在能做什么”，先不要进入下一页，优先补那一页的答卷和复盘。
