Agent 方向

先理解 Agent 怎么跑，再把它做成可靠系统。

这个方向围绕“让模型持续做事”展开：基础课讲循环、工具和状态；Context Engineering 管理每一步该看什么；Harness、工具契约和可靠性课程处理执行、恢复、评测和追踪。

推荐路径

ROUTES

刚开始理解 Agent

Agent 基础与工程 → 多智能体系统。先会判断单 Agent 是否足够，再谈拆分。

要做可复用工作流

Agent 基础与工程 → Agent Skills → Context Engineering。先有运行边界，再封装能力，再管理上下文。

要做可靠 Agent 系统

Agent 基础与工程 → Agent Harness Engineering → 工具调用契约 → Agent 可靠性工程。

要理解代码 Agent

Agent 基础与工程 → Claude Code 工作流 → Agent Harness Engineering / MCP。
