## 资料熔炼

### 领域术语表

| 术语 | 直觉解释 | 精确定义 / 论文语境 | 常见误解 |
|---|---|---|---|
| full fine-tuning | 改整台模型 | 下游任务训练时更新全部预训练参数，得到 `Phi_0 + Delta Phi` | 以为只要效果最好就没有部署代价 |
| PEFT | 只训练一小部分任务参数 | parameter-efficient adaptation / tuning，冻结大部分预训练权重，只学习小型任务参数 | 以为参数少一定效果差，或一定零延迟 |
| LoRA | 用两个小矩阵表达权重改变量 | 冻结 `W_0`，训练 `A` 与 `B`，令 `Delta W = BA` | 以为 LoRA 是新加一层顺序网络 |
| rank `r` | 给更新矩阵多少条任务方向 | `B in R^{d x r}`，`A in R^{r x k}`，`r << min(d,k)` | 以为 `r` 越大一定越好 |
| `alpha / r` | 控制 LoRA 分支输出尺度 | 论文中用 `alpha/r` 缩放 `Delta W x`，减少改变 `r` 时重调超参的负担 | 以为它就是学习率，或可以随便省略 |
| merge | 推理前把更新并回原权重 | 部署时显式存 `W = W_0 + BA`，推理仍走原线性层 | 以为训练时也必须 merge |
| `W_q, W_k, W_v, W_o` | 注意力里的四类投影矩阵 | self-attention 中 query/key/value/output projection | 以为 LoRA 一定要加到所有矩阵 |
| intrinsic rank | 任务更新真正需要的低维方向 | 论文假设适配更新 `Delta W` 的有效方向数很小 | 以为所有任务都只要 `r=1` |
| adapter | 插入模型中的小模块 | 在 Transformer block 中顺序插入瓶颈层 | 以为参数少就不会增加推理延迟 |
| prefix tuning | 学可训练提示向量 | 优化特殊 token 的 embedding 或层激活 | 以为增加 prefix token 不会占上下文长度 |

### 知识骨架

1. **为什么需要 PEFT**：大模型全量微调在存储、显存、任务切换和部署上变得昂贵；已有 adapter/prefix 方法解决一部分问题，但引入延迟或占用序列长度。
2. **LoRA 的公式**：把权重更新 `Delta W` 写成 `BA`，冻结 `W_0`，只训练 `A` 和 `B`；理解形状、参数量、初始化、缩放和 merge。
3. **放进 Transformer**：LoRA 原则上可用于任意 dense layer，论文主要研究 attention 权重，常用 `W_q` 与 `W_v`；工程收益来自显存、checkpoint、吞吐和任务切换。
4. **实验读法**：比较 FT、BitFit、prefix、adapter、LoRA，在 RoBERTa/DeBERTa/GPT-2/GPT-3 上读参数量和任务指标，不把单一表格读成绝对定律。
5. **低秩解释**：rank 扫描、子空间相似度、`Delta W` 与 `W` 的关系，解释为什么小 `r` 可行，也说明边界。
6. **实践验收**：把 LoRA 放到真实项目中时如何选 target modules、rank、alpha、merge、保存与排错；能复述论文贡献与限制。

### 可练习点

- 根据模型尺寸、层数、target modules、rank 计算 LoRA 可训练参数量。
- 读懂 `h = W_0 x + BAx`，解释每个矩阵的形状和数据流。
- 区分 adapter、prefix tuning、BitFit、full fine-tuning、LoRA 的训练位置、推理影响和适用边界。
- 从实验表格判断：LoRA 的优势是否来自参数量、延迟、上下文长度、模型质量或工程部署。
- 用 rank 实验和子空间相似度解释“低秩有效”而不夸大成“rank 越小越好”。
- 诊断常见实现错误：冻结失败、形状反了、初始化错误、merge 时重复叠加、target module 选错。

### 资料边界

- 论文材料覆盖 LoRA 的提出动机、数学形式、Transformer 应用、主要实验和低秩现象分析。
- 论文没有覆盖现代 PEFT 生态的所有后续变体，例如 QLoRA、DoRA、AdaLoRA、LoRA+、rsLoRA 等；这组资料只在最后一块提示实践边界，不扩写成后续综述。
- 实验依赖 2021 年论文中的模型和任务。学习时应关注论证结构和方法机制，不把具体数值当作 2026 年所有模型的默认结论。
- 论文里的部分 PDF 图像未嵌入 HTML；学习页用 SVG 和表格重建核心关系，保留来源说明。
