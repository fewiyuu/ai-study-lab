- 练习记录导出（Markdown）
- 参考资料与来源标注

## 使用方式

1. **打开网站首页**，选一条学习路线或一门课程
2. **进入课程索引**，选子课程 → 选单元学习页
3. **用搜索**（顶部导航栏 🔍 或 [search.html](search.html)）直接找内容

学习页可直接在浏览器阅读，也可下载 Markdown 源文件放入自己的笔记工具。

## 关于

- 以中文为主，保留英文术语方便查官方文档
- 内容标注来源，不包装成权威教材
- 持续整理中，链接和入口随学习进度迭代
- 仅含公开学习资料，不含日记、隐私或商业内容
