# 30_研究

这里存放研究笔记、分析、复盘和专题梳理。

## 适合放在这里的内容

- 工具或方案对比
- 复盘类文档
- 学习主题的阶段性总结
- 面向项目决策的背景研究

## 建议做法

- 尽量写明来源、日期和研究主题
- 研究结果尽量链接到项目或知识库
- 长期稳定的结论可以继续提炼到 `40_知识库`
- 学习资源默认中文优先：中文教程 / 中文文档 > 双语或中文版官方资料 > 英文原文
- 只有在没有合适中文资料、需要查官方 API、排查报错或确认最新规则时，才把英文资料放到主线，并写明原因
