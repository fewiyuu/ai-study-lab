类神经网络训练不起来怎么办：局部最小值与鞍点

P03 把 optimization failure 放进诊断树。P04 讨论一种常见解释：训练过程中 loss 不再下降，gradient 变得很小，参数不怎么更新。很多人会马上说“卡在 local minima”，但这句话太粗。

Gradient 为零的地方统称 critical point。它可能是 local minima，也可能是 local maximum，更常被讨论的是 saddle point。Saddle point 附近有些方向 loss 变高，有些方向 loss 变低，所以理论上仍有路可以走。

  1. 解释 gradient 为零为什么会让 gradient descent 停住。
  2. 区分 critical point、local minima、local maximum、saddle point。
  3. 用 Hessian eigenvalue 的正负判断局部地形。
  4. 说明负 eigenvalue 对应的 eigenvector 为什么能指出下降方向。
  5. 解释为什么深度学习里不要随便把训练失败归咎于 local minima。

本节的概念地图

map

训练停住的表面现象是 loss 不再下降。更具体一点，是参数更新变得很小。Gradient descent 的更新量来自 gradient，如果 gradient 接近零，更新步伐自然也接近零。问题是：gradient 接近零只告诉我们“当前点一阶斜率小”，还没有告诉我们这个点是不是最低点。

要判断点附近的地形，需要看二阶信息，也就是 Hessian。Hessian 的 eigenvalue 全正，附近各方向大多往上，是 local minima；全负，附近各方向往下，是 local maximum；有正有负，说明有些方向上升、有些方向下降，就是 saddle point。

loss 停住→ gradient ≈ 0→ critical point→ 看 Hessian→ local minima / saddle

直觉

一阶斜率告诉你当前方向陡不陡，二阶曲率告诉你附近地形怎么弯。

正式说法

在 critical point，gradient 为零，局部二阶近似由 Hessian 的二次型决定。

判断句

“gradient 为零”不是“到达 local minima”的充分证据。

训练为什么会停住

gradient descent

Gradient descent 的核心更新是 θ ← θ - ηg。θ 是全部参数，η 是 learning rate，g 是 loss 对参数的 gradient。如果 g 很小，更新量 ηg 就很小；如果 g = 0，这个公式直接告诉你参数不会动。

训练停住不一定代表模型已经足够好。你可能看到 training loss 卡住，但 loss 仍然很高，深模型甚至比浅模型更差。这时就回到 P03 的 optimization failure：模型有能力表达更好函数，但优化没有找到。

过去常见说法是“卡在 local minima”。这句话的问题是把所有 gradient 为零的情况都叫 local minima。更准确的说法是：训练可能卡在 critical point，接下来要判断它是哪一种 critical point。

θt+1 = θt - η ∇L(θt)
当 ∇L(θt) = 0 时，参数不会因为这条更新式继续移动。

现在你应该能做什么

看到 loss 停住时，先检查 gradient norm，而不是直接说模型过拟合或卡 local minima。

Critical point：gradient 为零的总称

critical point

Critical point 是一类点，不是一种地形。它只要求 gradient 为零。Local minima 是 critical point 的一种：附近每个方向 loss 都不比当前低。Local maximum 也是 critical point：附近每个方向 loss 都不比当前高。Saddle point 也是 critical point：有些方向往上，有些方向往下。

Saddle point 的名字来自马鞍形状。沿一个方向看，它像谷底；沿另一个方向看，它像山顶。所以你只看一维或二维切片，很容易误判。深度学习参数空间维度很高，某个低维图上像 local minima 的位置，在更高维方向上可能仍有下降路径。

判断提示

一个点是不是 saddle，不看单一方向。要问附近是否同时存在上升方向和下降方向。

Critical point 的分类要靠局部地形，而不是靠 loss 停住这个现象本身。

Hessian：用二阶信息判断地形

hessian

完整的 loss surface 很复杂，我们通常画不出来。但在某个参数点附近，可以用 Taylor 展开做局部近似。这个近似里有两类信息：gradient 是一阶项，Hessian 是二阶项。到 critical point 时，gradient 项消失，Hessian 决定附近地形。

Hessian 是由二阶偏微分组成的矩阵。你不需要手算大型网络的 Hessian，但要理解它的判断规则。把 Hessian 的 eigenvalue 看成各主方向上的曲率：正值表示沿那个方向像碗底，往外走 loss 增大；负值表示沿那个方向像山顶，往外走 loss 减小。

L(θ) ≈ L(θ₀) + gᵀ(θ-θ₀) + 12(θ-θ₀)ᵀ H (θ-θ₀)
在 critical point，g = 0，所以主要看 Hessian 的二次型。

看到 Hessian eigenvalue 的正负时，能判断 critical point 属于哪类地形。

负 eigenvalue 指出逃离 saddle 的方向

escape

如果 Hessian 有负 eigenvalue，它对应的 eigenvector 就是一个下降方向。直觉上，这个方向的曲率是负的，沿着它走一点，二阶近似会让 loss 变小。也就是说，虽然 gradient 在 saddle point 为零，不再告诉你往哪里走，但 Hessian 仍能指出一条路。

这就是 saddle point 没有 local minima 那么可怕的原因。Local minima 附近所有方向都上升；saddle point 附近至少有一个方向下降。理论上，你可以沿负 eigenvalue 的 eigenvector 更新参数，离开 saddle point。

不过，真实深度网络参数很多，Hessian 非常大。完整算 Hessian 和 eigenvector 的成本通常太高，所以实际训练里很少直接用这种方法逃离 saddle。课程在这里讲它，是为了建立判断：卡住不等于无路可走，saddle point 可能还有下降方向。

若 H u = λu 且 λ < 0，则沿 u 方向的小移动会降低二阶近似下的 loss。
最小示例：用 Hessian 判断点类型python
import numpy as np

def classify_critical_point(H):
    eigvals = np.linalg.eigvalsh(H)
    if np.all(eigvals > 0):
        return "local minima"
    if np.all(eigvals < 0):
        return "local maximum"
    return "saddle point"

H = np.array([[0, -2], [-2, 0]])
print(classify_critical_point(H))  # saddle point: eigenvalue 有正有负

高维空间里，local minima 不一定是最大麻烦

high dimension

二维图很容易让人误会。你在一个二维切片上看到像谷底的地方，可能只是因为你只看了两个参数方向。真实网络有百万、千万甚至上亿参数，参数空间维度极高。多一个维度，就多一个可能下降的方向。

这也是为什么现代深度学习里，随口把训练失败归咎于 local minima 会显得不严谨。很多时候，saddle point、plateau、梯度消失、学习率问题、初始化问题、网络结构问题都可能让优化困难。P4 只处理其中一个角度：critical point 的地形分类。

你以后看到训练 loss 卡住，可以更精确地说：优化遇到了停滞，可能是 critical point、平坦区域、梯度尺度问题或训练设置问题。要继续排查，而不是一句 local minima 结束。

低维图的误导

二维切片里像 local minima，不代表高维空间没有下降方向。

高维直觉

参数越多，可走方向越多。saddle point 在高维讨论中很重要。

最小代码：检测 gradient norm 与 Hessian 类型

code reading

这段代码展示一个最小排查逻辑：先看 gradient norm 是否接近零；如果接近零，再看 Hessian eigenvalue 判断 critical point 类型。真实深度网络很少完整这样做，但这段代码能帮你把概念落到可执行检查上。

正确示例：critical point 的最小诊断python
import numpy as np

def diagnose_point(grad, hessian, eps=1e-6):
    grad_norm = np.linalg.norm(grad)
    if grad_norm > eps:
        return "还不是 critical point，gradient 仍可指方向"

    eigvals = np.linalg.eigvalsh(hessian)
    if np.all(eigvals > 0):
        return "critical point: local minima"
    if np.any(eigvals < 0) and np.any(eigvals > 0):
        return "critical point: saddle point，有下降方向"
    return "需要进一步检查，可能是平坦或退化情形"
错误示例：只看 gradient norm 就下结论python
if grad_norm == 0:
    print("卡在 local minima")  # 错：也可能是 saddle point 或 local maximum
