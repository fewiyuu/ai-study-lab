# 上下文化向量：同一个字，语境不同向量不同 {#embedding}
<!-- tag: contextualized embedding -->

BERT 输入一串 token，输出每个 token 的向量。这个向量叫 embedding。传统静态 word embedding 中，同一个词通常只有一个向量；BERT 不同，因为它通过 self-attention 看上下文。同一个“果”出现在“苹果汁”和“苹果股价”里，输出向量可以不同。

如果收集很多含“苹果”的句子，把每个“果”的 BERT embedding 拿出来算 cosine similarity，就会发现食物语义的苹果彼此更接近，公司语义的苹果彼此更接近，两组之间相似度较低。这个现象支持了常见说法：BERT 的输出向量携带当前上下文中的词义。

# 为什么填空能学语义：词义来自上下文 {#why}
<!-- tag: distributional hypothesis -->

语言学里有一个常见假设：一个词的意思可以从它常出现的上下文看出来。食物意义的苹果附近常出现“吃、果汁、进口、季节”；公司意义的苹果附近常出现“手机、专利、股价、发表”。上下文不同，语义就不同。

BERT 的 MLM 正好逼模型做这件事。把一个 token 遮住，模型必须从周围词汇抽取信息，才能猜出被遮住的内容。被遮住位置的输出向量可以看成上下文信息的压缩结果。若它能预测正确 token，说明它已经捕捉到某些语义和语法线索。

这个思想在 BERT 之前就出现过。CBOW 等 word embedding 方法也是从上下文预测中心词，只是模型较浅，计算条件也不同。BERT 可以看成更深、更强、能根据上下文动态改变表示的版本。

$$
Meaning is shaped by context.
 MLM forces the model to use context.
 Therefore the hidden vector tends to encode contextual meaning.
$$

# 奇怪实验：BERT 不只是在“看懂文章” {#strange}
<!-- tag: strange experiments -->

课程展示了一个让人困惑的实验：把 DNA、蛋白质或音乐序列映射成英文词汇，再送进英文预训练 BERT 做分类。比如 DNA 的 A/T/C/G 随机对应到英文词，得到一串不知所云的“句子”。直觉上，BERT 不应该看懂这些乱句。

但实验中，使用预训练 BERT 仍然可能比不使用 BERT 效果更好。这不一定否定 BERT 学到语义；它只是提醒我们：BERT 的好处可能不完全来自“理解自然语言语义”。它也可能提供了某种对深层模型训练有利的初始化，或者保留了适合序列分类的结构偏置。

这个现象的价值在于让我们保持科学谨慎。模型解释不能只讲一个顺口故事。看到 BERT 有用，应区分“我们观察到什么”和“我们确信原因是什么”。后者常常还需要更多实验。

观察乱映射 DNA/蛋白质/音乐后，预训练 BERT 仍可能有帮助。
可能解释语义以外，还可能是好初始化或训练动态优势。
学习态度不要把单一解释当完整真相。

# Multilingual BERT：跨语言迁移为什么神奇 {#multi}
<!-- tag: multilingual -->

Multilingual BERT 用多种语言一起做填空训练。Google 版本使用过 100 多种语言。神奇现象是：用英文 QA 数据 fine-tune 后，模型可以直接测试中文 QA，而且有不错表现。它没有看过中文 QA 标注，却能把英文任务能力迁移到中文。

一个直觉解释是跨语言 alignment。不同语言中意思相同的词，embedding 可能靠近。例如 rabbit 和 兔子、jump 和 跳，虽然语言不同，但在多语言 BERT 空间中可以有相近表示。这样模型学到的 QA 任务头不只适用于英文向量，也能作用在中文向量上。

不过 alignment 不是自动保证。课程提到自己训练较小资料量的多语言 BERT 时，alignment 效果并不好；把每种语言资料量增大后，loss 才明显下降，跨语言对齐才出现。这说明很多神奇现象需要足够数据和算力才会显现。

## box warn
### 数据量门槛
小数据未必学出对齐；规模变大后现象才明显。

# 语言差向量：意思靠近，但语言身份还在 {#vector}
<!-- tag: language vector -->

如果 multilingual BERT 真的把不同语言完全混在一起，它可能不知道该用哪种语言填空。但实际上它给英文句子填英文，给中文句子填中文，说明语言身份没有消失。

课程中的实验是：把所有英文 token 的 embedding 平均，把所有中文 token 的 embedding 平均，两者相减，得到一个“英文到中文”的差向量。给模型读英文句子后，把这个差向量加到 embedding 上，再让模型做填空，模型会输出混合但偏中文的结果，例如把 no one、me 等部分转成中文语义。

这个现象很有意思：同义词跨语言可以靠近，但语言信息可能藏在整体偏移方向里。它不是完美翻译，但说明 embedding 空间里有可以被操作的语言结构。

# 最小例子：如何观察 BERT embedding {#code}
<!-- tag: minimal example -->

下面伪代码展示本节实验的形状：取同一个词在不同句子里的 hidden vector，再算相似度。

正确示例：同词不同语境的相似度pseudo python

```text
sentences = [
 "今天买了苹果来吃",
 "苹果股价下跌",
 "苹果手机发布新品"
]
vectors = []
for s in sentences:
 hidden = bert(s)
 vectors.append(hidden[position_of("果")])

similarity_matrix = cosine_similarity(vectors, vectors)
```

错误示例：把 token id 相同当作语义相同pseudo

```text
# 错误理解：两个“果” token id 一样，所以 BERT 输出向量一定一样。
v1 = bert("苹果汁")[pos]
v2 = bert("苹果股价")[pos]
assert v1 == v2

# 修正：BERT 输出是 contextualized embedding，会受上下文影响。
```

# 常见误区与修正 {#mistakes}
<!-- tag: pitfalls -->
<!-- layout: grid2 -->

## box redsoft
### 误区 1：同一个 token 的 BERT 向量永远一样
错因：混淆 token id 和 contextualized embedding。后果：看不懂一词多义。修法：BERT 输出受整句上下文影响。

## box redsoft
### 误区 2：BERT 有用只因为它懂语义
错因：把常见解释当完整解释。后果：无法解释 DNA/音乐等奇怪实验。修法：考虑好初始化、结构偏置和训练动态。

## box redsoft
### 误区 3：乱映射实验证明 BERT 不懂语言
错因：过度反推。后果：否定了语义聚类证据。修法：它只说明成功原因不止一个。

## box redsoft
### 误区 4：多语言 BERT 完全抹掉语言差异
错因：只看到跨语言对齐。后果：无法解释它仍用正确语言填空。修法：语义可对齐，语言身份仍可存在。

## box redsoft
### 误区 5：小数据也一定能学出 multilingual alignment
错因：忽略规模门槛。后果：复现实验失败时误判方法无效。修法：注意资料量、训练时间和算力。

## box redsoft
### 误区 6：Embedding 空间操作等于完美翻译
错因：把现象当成产品能力。后果：高估模型可控性。修法：它展示结构，但输出可能是混合语言和粗糙结果。

## box soft
### 现在你应该能做什么
用“现象、可能解释、证据边界”三步讨论 BERT 的能力，而不是只讲神奇故事。

# 术语与检查：解释 BERT 要保留边界 {#terms}
<!-- tag: terms / checklist -->

P24 的重点不是猎奇，而是训练解释习惯：先描述观察到的 embedding 现象，再提出可能机制，最后说明证据边界。BERT 确实表现出语义结构，但它为什么在奇怪序列上也有用，不能只用“懂语义”一句话解释。

| 术语 | 一句话解释 | 易混点 |
| --- | --- | --- |
| Contextualized embedding | 同一个 token 的输出向量会随上下文改变。 | token id 相同不代表最终 hidden vector 相同。 |
| Cosine similarity | 用向量角度衡量两个 embedding 是否接近。 | 相似度聚类是现象，不自动给出完整因果解释。 |
| Good initialization | 预训练权重可能让下游优化起点更好。 | 它可以和语义解释同时成立，不是互相排斥。 |
| Multilingual alignment | 不同语言的同义表示在空间中靠近。 | 语义对齐不等于语言身份完全消失。 |

## box soft
### Worked example
收集“苹果汁、苹果茶、苹果股价、苹果手机”等句子，取每个“果”的 BERT 向量并算 cosine similarity。若食物语境彼此接近、公司语境彼此接近，说明 contextual embedding 捕捉了语义差异；但这不能单独解释 DNA 随机映射实验。

### 小检查表
- 你说的是观察、解释，还是已经验证的因果？

- 多语言实验是否同时说明 alignment 和语言身份？
- embedding 空间操作是否被过度解读成完美翻译？
