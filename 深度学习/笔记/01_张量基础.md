# 01 · 张量基础

## 张量就是 PyTorch 版 NumPy

```python
torch.tensor()  ≈ np.array()
torch.zeros()   ≈ np.zeros()
torch.ones()    ≈ np.ones()
torch.eye()     ≈ np.eye()
```

`.shape` 同样是最重要的属性。`*` 和 `@` 规则和 NumPy 一样。

## NumPy ↔ Tensor 互转

```python
t = torch.from_numpy(arr)   # NumPy → Tensor（共享内存）
arr = t.numpy()              # Tensor → NumPy（共享内存）
```

> [!warning] `.numpy()` 是张量的方法，不是 `torch` 模块的函数
> `t.numpy()` ✅ / `torch.numpy(t)` ❌ ——不加参数，不加括号里的东西

## 最小代码

```python
import torch

t = torch.tensor([[1, 2], [3, 4]])
print(t.shape)          # torch.Size([2, 2])
print(t @ t.T)          # 矩阵乘法
print(t.sum(), t.mean())
```

> [!tip] 设备
> `t.to("cuda" if torch.cuda.is_available() else "cpu")`——学习阶段 CPU 够用。
